Welcome to Group 10's Web Application Development 1 and Responsive Web Design Project

Our midterm project for Web Application Development 1 and Responsive Web Design is a team-based task where we work together to design and build a simple but functional web application using HTML, CSS, and minimal JavaScript. The goal of this project is to apply what we have learned in front-end development, practice collaboration, and create a polished final product that follows proper coding structure, good design, and user-friendly features. This project also helps us improve teamwork, as each member has a role in planning, designing, coding, and testing to make sure the website is complete and well-presented.

This project showcases our skills and achievements in web application development. Explore our profiles to learn more about each member of our group BY CLICKING THE PROFILES BELOW.
